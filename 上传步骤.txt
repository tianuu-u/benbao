# 上传到 GitHub 的步骤（仓库名：benbao）

## 第一步：在 GitHub 上创建仓库

1. 访问 https://github.com/new
2. Repository name 输入：benbao
3. 选择 Public 或 Private（根据你的需要）
4. ⚠️ 重要：不要勾选 "Initialize this repository with a README"
5. 点击 "Create repository"

## 第二步：在终端执行以下命令

打开终端，复制粘贴以下命令（一行一行执行）：

cd "/Users/uutian/Desktop/未命名文件夹 2"

git init

git add .

git commit -m "Initial commit"

git branch -M main

git remote add origin https://github.com/你的用户名/benbao.git

git push -u origin main

⚠️ 注意：把 "你的用户名" 替换成你的 GitHub 用户名

## 第三步：启用 GitHub Pages

1. 在 GitHub 网站上打开你的 benbao 仓库
2. 点击 "Settings"（设置）
3. 左侧菜单找到 "Pages"
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main"，文件夹选择 "/ (root)"
6. 点击 "Save"
7. 等待几分钟，你的网站就会在 https://你的用户名.github.io/benbao 上可用

## 如果上传失败（文件太多）

可以分批上传：

git add index.html npc-login.js 字体/ 游戏截图/
git commit -m "Add main files"
git push

git add 农作物/
git commit -m "Add crop images"
git push

git add music/
git commit -m "Add music files"
git push
